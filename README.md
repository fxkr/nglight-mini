# nglight-mini r0

## What is it?

A tiny, I2C-controlled 6-channel low-side PWM LED driver based on the PCA9685.
It's just a smaller version of this: http://ng.uavp.ch/moin/Documentation/NGlight

Both schematic and layout are proven.
